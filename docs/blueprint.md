# **App Name**: FinShenz

## Core Features:

- Scenario List: Display a scrollable list of financial scenarios on the home screen.
- Scenario Explanation: Present a concise explanation of the financial concept in the 'Scenario Screen'.
- Interactive Checklist: Provide an actionable checklist in each scenario for users to complete.
- Progress Persistence: Save the checklist state locally on the device to persist progress.
- Points Calculation: Calculate and award points (100) upon completing a scenario checklist.
- Profile Update: Automatically update the user's profile (total points, topics mastered) upon scenario completion.
- Profile Screen: Show collected user's points and mastered topics

## Style Guidelines:

- Primary color: Cheerful blue (#64B5F6) to evoke trust and security.
- Background color: Light blue (#E3F2FD), a desaturated version of the primary, to keep a light and airy feel.
- Accent color: Yellow-green (#9CCC65) for positive reinforcement on task completion.
- Body and headline font: 'PT Sans', a humanist sans-serif, is used for both headlines and body, providing a modern look and some warmth for an encouraging and readable feel.
- Use simple, clear icons to represent financial concepts and actions within the checklists.
- Employ a clean and intuitive layout with clear visual hierarchy, ensuring ease of navigation and a user-friendly experience.
- Incorporate subtle animations for task completion and points updates to provide positive feedback to the user.
